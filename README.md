# HBU-Wiki 🏫

> 由个人发起、校友共建的河北大学维基百科，帮助大家了解学校、获取有效信息。

## 📌 项目状态

**目前框架已搭好**，新增了动画组件与快捷通道，正在逐步完善页面与内容填充。

所以你现在看到的每个板块基本都只有一篇占位内容，别急，会有的 🙃

## 🛠️ 技术栈

- [VitePress](https://vitepress.dev/) — 静态站点生成器
- 自定义主题 + CSS 变量设计系统
- 部署方式：GitHub Pages 自动部署

## ✨ 已完成的前端与动效优化

*   **小米 MiMo 科技极简美学**：全站重构为 MiMo 风格，亮色模式下纯白黑字，暗色模式下纯黑底色与深空灰卡片。卡片和代码块统一使用 `28px` 超大有机圆角，无渐变无阴影，极致扁平。
*   **物理阻尼跟随光晕 (桌面端)**：基于 Lerp 缓动算法的背景弥散光晕，流畅跟滑，鼠标移出自动渐隐；手机端自动关闭。
*   **滚动渐入特效 (Scroll Reveal)**：通过 `IntersectionObserver` 监控，首页各版块与卡片在滑入屏幕时平滑向上淡入，过渡舒适。
*   **数字滚动动态看板**：首页配置扁平化双列数据看板，视口触发 `requestAnimationFrame` 数字滚动增长特效。
*   **极简高反差胶囊按钮**：首页主按钮重构为纯黑（亮色）/纯白（暗色）圆角胶囊样式，悬停微缩放响应。
*   **高反差胶囊侧边栏**：侧边栏选中项升级为圆角胶囊高亮背景，文字反差对比显示，富有设计感。
*   **常用快捷通道/友情链接**：集成官网、教务、图书馆、校园网常用直达入口，支持 3D 浮动投影与外链图标滑动。
*   **探索主题标签栏**：快速跳转食堂测评、避坑指南、校园地图等高频百科页面，胶囊标签悬停微弹。
*   **动态鸣谢与贡献者**：自动拉取 GitHub Contributors 列表，支持头像 hover 放大、发光及贡献数徽章。
*   **原生级 App 手机端转场**：将页面过渡动画曲线升级为 `cubic-bezier(0.16, 1, 0.3, 1)` 并配以微缩放，手机端页面切换顺滑无缝。
*   **文档排版全面重构**：大标题、行内代码、提示横幅 (Callouts) 及极简扁平表格均按高端科技设计规范重构。
