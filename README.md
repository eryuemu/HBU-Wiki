# HBU-Wiki 🏫

> 由个人发起、校友共建的河北大学维基百科，帮助大家了解学校、获取有效信息。

## 📌 项目状态

**目前只是一个架子。** 框架搭好了，内容还很单薄，暑假会集中精力填充完善。

所以你现在看到的每个板块基本都只有一篇占位内容，别急，会有的 🙃

## 🛠️ 技术栈

- [VitePress](https://vitepress.dev/) — 静态站点生成器
- 自定义主题 + CSS 变量设计系统
- 部署方式：待定（暑假一起搞）

## ✨ 已完成的前端优化

- **明暗模式切换**：完整的亮色/暗色主题支持，一键切换
- **深色主题**：纯黑底色，校色蓝 `#1E56B0` 贯穿全站
- **首页 Hero 交互**：校徽 3D 鼠标追踪倾斜 + 光影反射效果
- **Feature 卡片**：Lucide 风格 SVG 图标，hover 上浮 + 入场依次延迟动画
- **页面过渡**：路由切换 fade + slide-up 过渡动画
- **导航栏**：毛玻璃效果，链接 hover 蓝色下划线展开动画
- **文档页**：标题、链接、表格、提示框全部按设计规范重写
- **响应式**：移动端适配
