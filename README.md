# HBU Wiki 🏫

> 由个人发起、校友共建的河北大学维基百科，致力于提供最真实、最接地气的校园生存与学术指南。

## 📌 项目状态

目前项目框架已完全搭建完毕，并完成了针对内容和结构的重大优化：

- **核心内容上线**：
  - **🚀 [转专业数据全解（2026年）](https://github.com/eryuemu/HBU-Wiki/blob/main/academics/transfer.md)**：已整理完 2026 届大一下学期转专业的官方政策，包含 67 个专业报录比数据、28 个专业最低绩点录取线梯队（T0-T4）、大学数学等级限制分析，以及贴吧楼主与小吧主关于新政策（平行志愿、仅看首学期通识通修水课、无笔试面试）的真实批判和实操建议。
  - **🌱 新生引导**：独立作为顶级板块，为新入学的同学梳理必备证件、行李建议与现场迎新流程。
- **界面与体验优化**：
  - 去除了主页冗余的 AI 占位组件（“百科板块”、“探索主题”、“常用通道”），使主页信息架构更佳纯粹。
  - 彻底完成了全站的“去 AI 幻觉”内容校对，删除了不确定和非官方印证的学术、综测、毕业等页面，修正了志愿平台（“志愿河北”）、统一身份认证等错误技术名词。
  - 修复了亮色模式下学校 Logo 与主页大校徽反色/隐形的问题，在白色背景下完美呈现高雅的“河大校色蓝” (`#1E56B0`)。

## 🤝 参与共建

这是一个开源项目，非常欢迎每一位河大校友和在校同学参与贡献！

1. Fork 本仓库：[https://github.com/eryuemu/HBU-Wiki](https://github.com/eryuemu/HBU-Wiki)
2. 在本地拉取代码并安装依赖：
   ```bash
   npm install
   ```
3. 启动本地开发服务预览：
   ```bash
   npm run docs:dev
   ```
4. 撰写或修改对应的 `.md` 页面，确认无误后提交 Pull Request。
