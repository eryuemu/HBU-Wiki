# 📚 图书馆指南

河大图书馆是学习的好去处，了解这些让借阅更高效。

## 馆舍分布

### 五四路校区图书馆

* **地址**：校园中心区域
* **开放时间**：周一至周日 8:00-22:00（假期另行通知）

### 七一路校区图书馆

* **地址**：校区中心位置
* **开放时间**：周一至周日 8:00-22:00（假期另行通知）

## 借阅规则

| 项目 | 规定 |
|------|------|
| 借阅数量 | 本科生 10 册，研究生 20 册 |
| 借阅期限 | 30 天，可续借一次 |
| 续借方式 | 登录图书馆系统在线续借 |
| 逾期处理 | 逾期需缴纳滞纳金 |

## 电子资源

* **中国知网（CNKI）**：校内 IP 直接访问
* **万方数据**：校内 IP 直接访问
* **维普期刊**：校内 IP 直接访问
* **Web of Science**：校内 IP 直接访问
* **校外访问**：通过 VPN 连接后使用

## 自习室

* 图书馆内设有多个自习室
* 考试周座位紧张，建议早到
* 部分自习室需要预约

## 其他服务

* **馆际互借**：可借阅其他高校图书馆的书籍
* **文献传递**：获取本馆没有的文献
* **查重服务**：毕业论文查重（具体流程见教务处通知）

::: tip 小贴士
借书前先在图书馆系统检索，确认馆藏位置，避免白跑一趟。
:::
