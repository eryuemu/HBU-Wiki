# 📅 校历与节点

了解河大学期安排，合理规划学习和生活。

## 学期制度

河北大学实行**两学期制**：

* **秋季学期**：9月初 ~ 次年1月中旬（约18周）
* **春季学期**：2月下旬/3月初 ~ 7月中旬（约18周）

## 重要时间节点

| 时间 | 事件 |
|------|------|
| 9月初 | 开学报到、军训（大一） |
| 10月 | 国庆假期 |
| 11月中旬 | 期中考试（部分课程） |
| 1月初 | 期末考试周 |
| 1月中旬~2月 | 寒假 |
| 3月初 | 春季开学 |
| 4月 | 清明假期 |
| 5月 | 五一假期、运动会 |
| 6月下旬 | 期末考试周 |
| 7月~8月 | 暑假 |

## 考试安排

* **四六级考试**：每年6月和12月
* **计算机等级考试**：每年3月和9月
* **期末考试周**：一般在学期最后1-2周

::: tip 提醒
具体校历以学校教务处官方发布为准，每学年开学前会公布当年校历。
:::
